Create deflationary token with taxes (MATIC)

Token with up to 2 taxes and automatic burning during every transfer.
For example, tax 1 is set to 2%, tax 2 is set to 1% and deflation is set to 1% (totally 4%); Alice sends 100 tokens to Bob, Bob receives 96 tokens (100-4%), 2 tokens (2% of 100) are sent to address for tax 1, 1 token (1% of 100) to address for tax 2, and 1 token (1% of 100) is burned.
